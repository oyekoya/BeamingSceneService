QStringList SceneInterface::sceneClientsVisible ( bool  IDonly = false)

Obtain a list of all the objects that are visible on the Beaming Scene Server.

Parameters:
IDonlyWhether to only return the GUID's (true - from Beaming Scene getPeersID()) or also obtain info about all connected (root) nodes on for each GUID (false - from Beaming Scene getNodesInfo())
Returns:
QStringList containing the result
If IDonly is true, each string is just the GUID
If IDonly is false (default), each string has the format 
GUID,NAME,NODEID,TYPE,CONFIG
where multiple published objects from the same client (eg. multiple tracked avatars/objects) will result in duplicates of the GUID with different names.

For obtaining a specific field from the full return format, one may use eg. QString::section(): 

QString name = avatarIds.at(1).section(",",1,1); // NAME
QString guidAvatar = avatarIds.at(1).section(",",0,1); // GUID,NAME
